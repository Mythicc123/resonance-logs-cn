# Changelog v0.0.3

## Logs

- Added display for buffs without icons
- More flexible UI settings for buff monitor, skill monitor, and resource bar. After enabling edit mode, you can freely position each icon-based buff, icon-less buff monitor bar, Skill CD bar, and resource bar. Exiting edit mode makes the overlay click-through, preventing interference with in-game UI clicks
- Buff timers now reference server time; the final recorded time is the offset-adjusted local time
